UX Process

1. User imports flashcards
    - User clicks the import flashcards button
    - PDF file or manual inserting
    - User reviews, edits, and confirms their flashcards
    - User names their flashcard folder
    - User saves their flashcard folder to their json
2. User learns flashcards
    - User chooses the flashcard folder
        - If the user does not have any flashcard folder it will prompt themm to create one and provide a shortcut to the import flashcards page
        - If the user does have flashcard folders but wants to make a new one while already in the learn page, there will be a + button that will give them a shortcut to the import flashcards page
    - User can view the flashcards, flip them, and have it read aloud
    - At the end of the flashcards folder, the user is prompted to play a game with that folder, continue learning, or quit to title screen
3. User plays the game after learn feature
    -User spawns in on a trail 
    -Enemy approaches 
    -User can choose attacking, cards, and items
        -User starts with 2 default cards 
            -Throw rock/sand
                -(-)10hp
            -Jab with stick
                -50% chance enemy misses next attack
                - (-)5 hp
        -User can attack
            -Punch
            
            
    
CARDS

Tail Whip - Basic physical attack where gator whips with his tail (range of damage, 10-35)
Gator Bite - Basic physical attack where the gator chomps on the enemy (Range of damage, 10-25)
Water Borrage- Upgraded version of water bubble, takes an extra turn to charge up but deals more damage
Scratch - Basic physical attack where gator scratches the enemy (Range of damage, 10-20)
Chud Attack - gator pulls out his laptop and builds a hackathon winning project, instakills enemy due to shock (Rare card)
Snot Bubble - Gator spits a bubble of snot at the enemy (Range of damage, 10-20,chance of poison)
Death Roll - Upgraded version of barrel roll, Gator rolls on the enemy dealing massive damage (Range of damage, 35-55)
Barrel Roll - Gator rolls out of the way of the enemy attack (50% chance to dodge next attack)
Water Bubble - Gator spits a bubble of water at the enemy (Range of damage, 10-25)
